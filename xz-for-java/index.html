XZ for Java

XZ for Java aims to be a complete implementation of XZ data compression in pure Java.

The project's main repository is on GitHub and is mirrored (with some delay) to Tukaani.

Check out the current source code:

git clone https://github.com/tukaani-project/xz-java

Features

XZ for Java supports single-threaded streamed compression and decompression and random access decompression. Threading is planned but has not been implemented.

  • Single-threaded streamed compression and decompression.
  • Random access decompression.
  • Delta, BCJ (x86, PowerPC, ARM, ARM-Thumb, IA-64, SPARC), and LZMA2 filters. The ARM64 filter is complete, but is not part of an official release yet.
  • All checksum types from the xz-file-format (CRC32, CRC64, SHA256).

Threading is planned but has not been implemented.
