XZ Projects

About

The projects XZ Utils, XZ Embedded, and XZ for Java are all maintained under the XZ Projects. Previously, they were maintained under the Tukaani Project, which was a Slackware based distribution. Nowadays the Tukaani Project no longer exists, but this work led to the XZ Projects.

XZ is a general-purpose data compression format. It is a streamable container format that allows for multiple filters, random-access reading, and a few different integrity check types. New filters are occasionally added to improve compression with specific types of data.
